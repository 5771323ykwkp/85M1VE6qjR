# Python图书管理系统 python28

## 项目概述

本项目是一个基于Python语言的图书管理系统，包括源码和基本的使用说明。系统采用Django框架进行开发，前端使用HTML、CSS和JavaScript进行界面设计，数据库使用sqlite。

## 技术栈

- 后端：Python + Django
- 数据库：sqlite
- 前端：HTML + CSS + JavaScript

## 功能模块

- 登陆功能
  - 用户登录与登出
- 图书管理
  - 图书信息添加、修改、查询和删除
- 借阅管理
  - 借阅记录添加、查询、更新和删除
- 个人管理
  - 个人信息查看与修改
- 会员管理
  - 会员信息管理
- 数据可视化图表
  - 系统数据的可视化展示

## 系统角色

- 系统管理员：负责整个系统的维护和用户管理
- 普通用户：可以进行图书借阅和个人信息管理

## 运行环境

- Python环境
- Django框架
- sqlite数据库

## 部署步骤

1. 配置Python环境
2. 安装Django框架
3. 配置sqlite数据库
4. 运行项目

## 目录结构

```
project/
│
├── apps/                # 业务逻辑目录
├── db/                  # 数据库文件目录
├── media/               # 静态文件目录
├── static/              # 前端静态文件目录
├── templates/           # 前端模板文件目录
├── manage.py            # 项目管理脚本
└── project/             # 项目配置目录
```
